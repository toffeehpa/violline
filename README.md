# Violline - VPN Vless Client

Modern VLESS client for Android powered by xray.

## Features

### Current

- UI built with Jetpack Compose
- Android VPN Service integration
- API 26+ support
- sing-box core integration
- Dark theme

### Planned

- VLESS support
- VMess support
- Trojan support
- Subscription management
- QR code import
- Configuration import/export
- Traffic statistics
- Multi-profile support

## Tech Stack

- Kotlin
- Jetpack Compose
- Android VPN Service
- sing-box

## Requirements

- Android 8.0 (API 26) or newer
